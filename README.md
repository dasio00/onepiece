# 원피스 데이터 탐색기

`index.html`을 브라우저로 열면 바로 사용할 수 있는 정적 웹앱입니다.

## 웹에서 수정하기

왼쪽 메뉴의 `수정` 탭에서 아래 내용을 바로 추가, 수정, 삭제할 수 있습니다.

- 인물: 이미지, 별명, 좋아하는 것, 인물 설명, 현상금, 악마의 열매, 조직/세부 조직
- 인물 추가 필드: 생일, 패기(무장색/견문색/패왕색)
- 인물 이력 필드: 키 이력, 현상금 이력, B-W-H 이력
- 에피소드: 권, 화, 화 제목, 간략한 내용, 등장 인물, 나온 기술
- 기술: 사용자, 메모
- 악마의 열매: 동물계/자연계/초인계, 동물계 구분(일반종/고대종/환수종), 모델, 각성 유무, 현재 능력자, 선대 능력자, 설명
- 조직: 해적/혁명군/해군/기타 아래에 세부 조직 추가
- 그룹: 여러 사람 선택, 그룹 이름, 그룹 설명
- 연표: 인물별 년도/내용 입력, 통합 연표에서 같은 연도 사건 묶기
- 카드 퀴즈: 입력된 데이터에서 나이, 키, 현상금, 혈액형, 출신지, 별명, 좋아하는 것, 악마의 열매, 조직, 연표 퀴즈 자동 생성
- 출신지: 이스트 블루/웨스트 블루/사우스 블루/노스 블루/그랜드라인 아래 국가 분류
- 등장편: WT100 공식 사이트의 `登場した編` 이름 목록을 한국어 이름과 일본어 원문으로 정리

## 에피소드 보기

왼쪽 메뉴의 `에피소드`에서 권을 선택하면 그 안의 화 목록이 나옵니다.
각 화를 누르면 화 제목, 간략한 내용, 등장 인물, 나온 기술이 함께 표시됩니다.
사람/기술 상세 화면의 화수는 에피소드 데이터에서 자동으로 계산되며, 작은 숫자 칩으로 표시됩니다. 숫자를 누르면 해당 에피소드로 이동합니다.
에피소드 상세의 등장 인물 이름을 누르면 그 사람 상세로 이동합니다.

## 연표 입력 형식

인물 수정 화면의 `연표`에서 `년도`와 `내용`을 각각 입력합니다.
왼쪽 메뉴의 `연표 > 통합 연표`는 같은 년도의 사건을 한곳에 묶어서 보여줍니다.

## 데이터 관리

저장한 내용은 브라우저 저장소에 남습니다. `수정 > 데이터 관리`에서 JSON으로 내보내거나 다시 불러올 수 있습니다.

## 직접 파일로 채우기

기본 예시 데이터는 `data.js`에 있습니다. 처음부터 파일에 넣고 싶으면 이 파일을 수정하면 됩니다.
